Ejemplo de Control de LEDs

  • Programación de Patrones de Luz
    • Control básico de GPIO con Python
    • Uso de la librería gpiozero
    • Creación de secuencias de luces
  • Ejemplo de Código 
    from gpiozero import LED
from time import sleep

led = LED(17)

while True:
    led.on()
    sleep(1)
    led.off()
    sleep(1)
